# FluentFlow 桌面端：功能与信息架构（基于 design/方案1-10 综合）

目标：在不大改现有 Header / Body 框架的前提下，把产品收敛成「敢开口」的口语核心，同时把错误沉淀为可复习资产，并在后续对话中自然带入复习计划。

## 核心闭环（唯一主线）
- Talk（沉浸式主动聊天）→ Capture（无打断纠错 + 成卡）→ Review（自动复习队列）→ Return（复习内容带入下一次对话）

## 顶层导航（桌面端）
- Talk：AI 主动聊天（默认入口）
- Review：SRS 复习中心（到期优先、少选择）
- Shadow：听力精听 + 影子跟读 + 发音纠正
- Live Proxy：真人聊天代理（你的原话 → AI 润色 → 对外发送）
- Library：素材库（文章/字幕/播客/网页剪藏/电影片段台词）
- History：会话与总结（每次会话的目标、错误、进步）
- Settings：声音/口音/隐私/快捷键

## Talk（核心）
- AI 主动引导：用「问题树」推进对话，不让用户冷场
- 场景化沉浸：酒店入住 / 面试 / 旅行 / 会议 / 电影对白复刻；支持场景背景与道具词
- 声音实验室：地区口音、音色、语速、情绪强度；同一语句多口音对比训练听力
- 无羞耻纠错：纠错不抢话、不弹窗打断；以「浮层小卡」轻轻出现，聊完再集中复盘
- 即时成卡：新词/短语/纠错句自动变成词卡/句卡/纠错卡，来源可追溯（哪段对话/哪句台词）
- 会话收束：结束自动生成“学习摘要 + 下次目标 + 推荐复习队列”

## Review（SRS 复习中心）
- 到期优先：默认只让用户做一件事——开始到期复习（极简坚持）
- 卡片类型：词卡 / 短语卡 / 句卡 / 纠错卡 / 发音卡
- 复习模式：拼写 / 听写 / 跟读评分 / 造句替换 / 快速选择
- AI 解释面板：错题原因 → 规则提示 → 1 题针对练习（不把用户带离复习流）
- 统计最少但有用：连胜、今日完成、薄弱点 Top3、遗忘曲线概览

## Shadow（精听/跟读/发音纠正）
- 精听播放器：波形、AB 循环、慢速、降噪、分句字幕对齐
- 影子跟读：逐句录音评分（准确度/流利度/重音），可视化对齐（目标波形 vs 你的波形）
- 句子库：收藏高频句，转为替换练习/仿写练习/角色扮演延展

## Live Proxy（真人聊天代理）
- 双通道界面：你的原话（可卡壳）与对外发送（AI 润色）并排显示，降低社交压力
- 实时纠音纠错：对你说的话实时提示（不发给对方），必要时自动修正发音易错词
- 代聊模式：你沉默/卡壳时，AI 可以接管 1 句（可配置强度与可见性）
- 语气控制：礼貌/坚定/幽默/正式，一键切换

## Library（素材库）
- 导入：字幕/播客/网页/文档；自动分句、分词、高亮生词
- 圈选即学：任何文本圈选 → 解释/例句/加入卡片/生成练习
- “电影对白”模式：经典台词片段 → 角色扮演 → 跟读评分 → 句卡沉淀
