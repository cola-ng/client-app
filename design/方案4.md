# 方案4：沉浸式对话模式｜边聊边记“即时词卡”

## 产品定位
- 人群：口语为主，希望“聊着聊着就学会”的用户
- 目标：把学习动作嵌在对话中：即时纠错 → 即时成卡 → 即时练

## 功能需求（核心）
- Chat：默认全屏沉浸，最少 UI 干扰
- 即时词卡（Floating Cards）：对话中出现的新词/短语自动浮出，可一键收藏
- 即时练习：收藏后立刻弹出 15 秒练习（拼写/跟读/替换造句）
- 对话总结：每次会话结束自动生成“学习摘要”和“下次目标”
- 单词查看：从浮卡点开进入详情

## 信息架构
- 立即对话（默认入口）
- 词库
- 复习
- 历史会话
- 设置

## 关键页面草稿图

### 1) 沉浸式 Chat + 浮卡
```
┌────────────────────────────┐
│ 主题: 旅行  计时 02:10  结束⏹ │
├────────────────────────────┤
│ 你: I want book a room.      │
│ 教练: ✅ I want to book a room.│
│      解释: want to + 动词原形  │
│                              │
│ 你: Is there any availability?│
│ 教练: Great! 更地道: Do you have…│
│                              │
│   ┌───────────────┐          │
│   │ 浮卡: availability  +收藏 │
│   │ 快练: 拼写/跟读/造句      │
│   └───────────────┘          │
├────────────────────────────┤
│ 输入…  🎙️   ➤   （长按句子=成句卡）│
└────────────────────────────┘
```

### 2) 会话结束总结（自动）
```
┌────────────────────────────┐
│ 本次会话总结                 │
├────────────────────────────┤
│ ✅ 你掌握了: book a room, reservation│
│ ⚠️ 易错点: want to, articles  │
│ 推荐下一步:                  │
│ - 复习 12 张卡片              │
│ - 角色扮演进阶(4min)          │
│ [去复习] [继续加练] [保存为脚本]│
└────────────────────────────┘
```

## 差异点
- “学习动作”嵌进对话流；适合把 Chat 做成主产品形态
