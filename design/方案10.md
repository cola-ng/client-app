# 方案10：极简专注｜一天三件事：聊、记、复习

## 产品定位
- 人群：反感复杂功能、只想坚持的用户
- 目标：把产品压缩成 3 个固定动作；用极简 UI 降低心理负担

## 功能需求（核心）
- 聊：每天 1 次对话（可语音）；结束自动总结
- 记：对话中一键收藏 5 个“今日词/句”
- 复习：只复习到期卡片（默认不让用户做太多选择）
- 单词查看：不做复杂页，只保留必要信息（释义+例句+发音+来源）
- 数据：只给“连胜 + 今日完成”

## 信息架构
- 今日（单页）
- 词库（轻量）
- 设置

## 关键页面草稿图

### 1) 今日单页（All-in-one）
```
┌────────────────────────────┐
│ 今日: 1/10  连胜 12 天         │
├────────────────────────────┤
│ ① 聊 (3分钟)                   │
│ [开始对话]                      │
├────────────────────────────┤
│ ② 记 (选5个词/句)               │
│ 今日已选: 2/5  +从对话中添加     │
├────────────────────────────┤
│ ③ 复习 (到期 18 张)             │
│ [开始复习]                      │
├────────────────────────────┤
│ 完成后奖励: 今日总结 + 明日建议   │
└────────────────────────────┘
```

### 2) 极简词卡
```
┌────────────────────────────┐
│ reservation  n.   🔊          │
│ 释义: 预订                     │
│ 例句: I'd like to make a reservation.│
│ 来源: 今日对话                  │
│ [标记掌握] [明天再说]           │
└────────────────────────────┘
```

## 差异点
- 选择极少、非常容易坚持；适合做“低摩擦日更”产品形态
