# 方案1：移动端底部导航｜学习仪表盘 + Chat 教练

## 产品定位
- 人群：碎片化学习者（通勤/排队），以“每天 10~15 分钟”为节奏
- 目标：用 Chat 驱动学习计划；词汇/句子自动沉淀到词库；每天完成闭环

## 功能需求（核心）
- Chat 教练：纠错、扩写、角色扮演、口语任务、总结错题
- 单词/短语查看：释义、例句、发音、同反义、词源/词根
- SRS 复习：新词/熟词、掌握度、错因标签（拼写/发音/用法）
- 听说训练：TTS 跟读、ASR 打分（准确度/连读/重音）
- 学习数据：连胜、目标、薄弱点雷达、时间分布

## 信息架构（底部导航）
- 首页
- Chat
- 词库
- 复习
- 我的

## 交互要点
- 任意对话消息长按：添加词卡/句卡/纠错卡
- 词卡支持“来源回溯”：点一下回到原对话上下文
- 复习页支持“1 分钟快速模式”

## 差异点（为什么选它）
- 上手最快；移动端学习闭环完整；适合先做 MVP
