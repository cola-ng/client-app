# 方案3：电影台词学习界面

## 设计目标
通过**经典电影对白**创造沉浸式学习体验，用户扮演角色，AI扮演对手戏。

## 交互细节

### 角色扮演流程
1. 用户选择电影和场景
2. 选择角色（Andy/Red等）
3. 显示角色信息、情感基调、表演指导
4. 开始角色扮演，AI扮演对手戏角色
5. 用户说话，AI智能回应，维持角色风格
6. 可随时查看情感提示、场景说明

### 跟读学习流程
1. 播放原声示范（可慢速）
2. 用户跟读录音
3. AI实时分析：语调、停顿、重音、清晰度
4. 显示波形对比，标注可改进处
5. 给出具体建议
6. 记录到学习点，自动加入复习计划

### 学习点收集
1. 从电影台词中自动提取新词/短语
2. 用户手动标记重要表达
3. 学习点关联到具体场景和情感
4. 在后续场景学习中自动复习

## 设计亮点

### 沉浸式学习
1. **真实场景** - 电影片段提供视觉+听觉的真实情境
2. **角色扮演** - 用户成为角色，而不只是"学习台词"
3. **情感共鸣** - 体会角色情感，语言学习更深层
4. **文化浸润** - 从经典电影中学习地道表达

### 专业分析
1. **多维度评估** - 语调、停顿、重音、清晰度、情感
2. **可视化对比** - 波形图直观显示差距
3. **具体建议** - 不只是分数，更有可操作指导
4. **进步追踪** - 历史数据展示学习曲线

### 智能辅助
1. **AI角色** - 真实扮演对手戏角色，维持对话连贯
2. **情感提示** - 引导用户表达正确情感
3. **自动记录** - 从对话中收集学习点
