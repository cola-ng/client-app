## 交互细节

### 纠错流程
1. 用户说出句子
2. AI实时分析（<1秒）
3. 检测到可改进点时，弹出温和提示
4. 显示原句、改进句、原因、示范
5. 用户选择：接受/跳过/学习
6. 接受后，自动加入复习计划

### 卡壳处理
1. 检测到沉默>5秒
2. 分析上下文和对话主题
3. 生成2-3个替身话术
4. 显示选项，用户选择
5. 可让AI代说，或自己模仿说
6. 记录卡壳点，后续针对性练习

### 口音切换
1. 切换时，AI立即应用新口音
2. 播放示范句子，让用户感受差异
3. 对比原口音 vs 新口音的波形
4. 用户可选择保存偏好口音

## 设计亮点

### 真实社交
1. **在线匹配** - 连接真实用户，不只是AI练习
2. **实时辅助** - AI在后台智能辅助，不打断交流
3. **卡壳救场** - 不让尴尬延续，自然接过话头
4. **社交目标** - 在真实对话中应用所学

### 温和支持
1. **隐式纠错** - 不直接指出错误，后台优化
2. **替身机制** - 卡壳时自然衔接，保护自尊
3. **进度鼓励** - 统计显示进步，保持动力
4. **匿名匹配** - 保护隐私，降低压力

### 智能增强
1. **多口音** - 一键切换，适应不同交流对象
2. **自动优化** - 语音、语调自动调整到最佳状态
3. **上下文理解** - 根据对话主题生成合适替身话术
4. **学习嵌入** - 在社交中自然复习，不占用额外时间
