# 方案1：场景沉浸式主界面

## 交互细节

### 场景切换动画
- 场景切换时，显示平滑过渡动画
- 例如：咖啡厅 ☕ -> 酒店 🏨，图标渐变切换
- 同时更新背景颜色/图案，营造氛围
- 播放环境音效（咖啡馆的咖啡机声、酒店的门铃声）

### 学习提示触发
- 检测到"可提升"的表达时，温和弹出提示
- 提示卡片从对话区下方滑入，3秒后自动收起
- 用户可选择"已知道"、"加入词卡"或"跳过"
- 加入词卡后，自动进入复习计划

### 进度可视化
- 学习进度条实时更新（分钟/句型/词汇）
- 每完成一个学习点，进度条前进一步
- 达到100%时，触发完成庆祝动画
- 自动生成今日学习报告

## 设计亮点

### 人性化
1. **温和支持** - AI教练用语友好，"放松点"、"错了没事"
2. **渐进提示** - 不是直接纠错，而是提供更好的表达
3. **场景代入** - 通过环境音效、氛围背景营造真实感
4. **即时反馈** - 每次输入都有回应，保持学习流

### 简洁性
1. **信息层次** - 主要操作突出，次要信息折叠
2. **视觉平衡** - 对话区占主体，工具箱在底部
3. **颜色克制** - 使用暖色调，避免视觉疲劳

### 创新性
1. **场景学习** - 不是单词/句子，而是"真实情境"
2. **AI陪伴** - Emma不只是工具，是学习伙伴
3. **智能复习** - 根据当日学习自动生成计划
4. **口音切换** - 一键切换不同口音，增强听力
